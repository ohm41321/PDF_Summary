# 🚀 Quick Start Guide / คู่มือเริ่มต้นอย่างรวดเร็ว

## 3 Steps to Get Started!

### Step 1: Setup (1 time only)
```
Double-click: setup.bat
```
This will automatically:
- ✓ Create virtual environment
- ✓ Install all dependencies
- ✓ Create required folders

### Step 2: Start LM Studio
1. Open LM Studio
2. Download & load a chat model (e.g., qwen2.5-7b-instruct)
3. Click "Start Server"

### Step 3: Run the App
```
Double-click: start.bat
```
The app will open automatically in your browser!

---

## 🎯 How to Use

1. **Upload PDF** → Click or drag-drop
2. **Get Summary** → Click "Generate Summaries"
3. **Ask Questions** → Type in chat box

---

## 💡 Pro Tips

- 🌐 Switch Thai/English with top-right button
- 📥 Export summaries & chat history
- 💡 Click suggested questions for quick start
- 🔍 Change models in the model selector

---

## ⚠️ Troubleshooting

**LM Studio not connecting?**
→ Make sure you clicked "Start Server" in LM Studio

**No models showing?**
→ Load a model in LM Studio first

**Upload fails?**
→ Make sure setup.bat completed successfully
